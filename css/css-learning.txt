class:
<style>

{
	text-align:center;
}
</style>

<p class="center">段落居中。</p> 

插入样式表的方法有三种:
外部样式表(External style sheet):<link rel="stylesheet" type="text/css" href="mystyle.css">

hr {color:sienna;}
p {margin-left:20px;}
body {background-image:url("/images/back40.gif");}

内部样式表(Internal style sheet)
<head>
<style>
hr {color:sienna;}
p {margin-left:20px;}
body {background-image:url("images/back40.gif");}
</style>
</head>

内联样式(Inline style)
<p style="color:sienna;margin-left:20px">这是一个段落。</p>
